# 📘 WAI Studio Design System (Zinc-Dark Edition)

WAI Studio는 **"Grey-Scale Density (무채색 밀도)"** 철학을 따릅니다. 

---

## 1. 레이어 관리 지침 (Layer Stratosphere Strategy)
모든 UI 요소는 임의의 숫자가 아닌, **지정된 고도(Altitude)** 규칙을 엄격히 따릅니다.

| Layer Level | Z-Index | 역할 및 포함 요소 |
| :--- | :--- | :--- |
| **L0 (Base)** | `10` | 배경 패널, 기본 레이아웃 바닥 |
| **L1 (Content)** | `20` | 캔버스 내부 객체, 트랙 클립, 영상 |
| **L2 (Sticky)** | `40` | 타임라인 헤더, 눈금자 (스크롤 시 고정) |
| **L3 (Toolbar)** | `100` | 상단 헤더, 프리뷰 툴바 (패널보다 위) |
| **L4 (Dropdown)** | `9000` | 메뉴 드롭다운, 컨텍스트 메뉴 (최상위) |
| **L5 (Overlay)** | `9999+` | 모달, 시스템 알림, 인스펙터 오버레이 |

---

## 2. 컴포넌트 디자인 규칙

### 🚫 메뉴 (Menus)
* **텍스트 전용:** 아이콘 없이 텍스트만 사용합니다.
* **하위 표시:** 하위 메뉴가 있는 경우 우측에 `>` 화살표를 표시합니다.
* **안전 영역 (Safe Bridge):** 메뉴와 버튼 사이의 간격으로 인해 닫히지 않도록 투명 영역(`::before`)을 확보해야 합니다.
